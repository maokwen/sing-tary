Tray scripts for sing-box

1. Prepare `sing-box.exe` and `config.json`.
2. Change executable path in `sing-box-service.xml`.
3. Download WinSW and rename it to `sing-box-service.exe`.
4. Run `sing-box-service.exe install` to install the service.
5. Run AHK2Exe to build `sing-tray.exe`.
6. Run `install-task.ahk` to register startup task.
